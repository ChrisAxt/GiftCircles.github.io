## Full Description

### App Store / Play Store Description

Stop the awkward moment when two people show up with the same gift. GiftCircles helps families and friend groups coordinate gift-giving for birthdays, holidays, and special occasions.

**How It Works**

Create an event for any gift-giving occasion. Invite your group with a simple join code. Members create wish lists and claim items they plan to buy. The app automatically hides claims from gift recipients, so surprises stay secret.

**Key Features**

Coordinate Group Gifting
- Create events for birthdays, holidays, weddings, and more
- Invite unlimited members via shareable join code
- See all wish lists in one organized place

Prevent Duplicate Gifts
- Claim items to show what you're buying
- Other members see claims instantly
- No more awkward duplicate presents

Keep Surprises Secret
- Gift recipients never see who claimed their items
- Privacy controls hide sensitive lists from specific people
- Claims are automatically filtered based on privacy rules

Stay Informed
- Push notifications for new lists and items
- Daily or weekly activity summaries
- Purchase reminders before event dates

Flexible List Options
- Create lists for yourself or others
- Add items with links, prices, and descriptions
- Support for multiple currencies
- Split expensive gifts with other members
- Random recipient assignment for Secret Santa style events
- Random receiver assignment for surprise item distribution

**Perfect For**
- Family holiday gift coordination
- Birthday party planning
- Wedding registries
- Secret Santa exchanges (with random assignment)
- Any group gift-giving occasion

**Free Features**
- Join up to 3 events
- Create unlimited lists and items
- Claim and track gifts
- Basic push notifications

**Pro Features**
- Unlimited event memberships
- Instant activity notifications
- Daily/weekly digest summaries
- Advanced notification preferences

Available in English, Swedish, German, French, Spanish, and Italian.
