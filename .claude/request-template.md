## Quick Templates for Common Requests

### Bug Fix Template
```
Bug: [Brief description]
Location: [File/function where bug occurs]
Steps to reproduce:
1.
2.
3.
Expected: [What should happen]
Actual: [What actually happens]
Error messages: [If any]
```

### Feature Request Template
```
Feature: [Name of feature]
User story: As a [user type], I want [goal] so that [benefit]
Acceptance criteria:
- [ ] Criterion 1
- [ ] Criterion 2
- [ ] Criterion 3
Related files: [Existing files that might be relevant]
```

### Refactoring Template
```
Code to refactor: [File/function/component]
Reason: [Why does this need refactoring?]
Goals:
- [ ] Improve readability
- [ ] Reduce complexity
- [ ] Improve performance
- [ ] Better maintainability
- [ ] Other: ___________
Constraints: [What must stay the same?]
```
